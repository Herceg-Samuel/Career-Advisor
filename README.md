# Career Advisor - Expert System

A rule-based system running in the browser using React. It uses Forward and Backward chaining to recommend careers based on user inputs.

## Knowledge Base

### Facts (Inputs)

- **Interests**: Technology, Art & Design, Science, Helping People, Business
- **Skills**: Coding, Drawing, Math, Communication, Management, Writing, Empathy

## Features

1.  **Forward Chaining (Explore)**: Select inputs to discover careers.
2.  **Backward Chaining (Validate)**: Select a target career to see if you qualify/what you are missing.
3.  **Conflict Resolution**: If multiple rules fire, the system prioritizes the most specific rule (most conditions).
